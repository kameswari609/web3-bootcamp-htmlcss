Summary : Latitude vs .X

The purpose of this project was to analyse how weather changes as you get closer to equator.to accomplish this analysis we first pulled out data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset ,we use MatPlotLib to plot various aspects of the weather vs Latitide. Factors we look at include : temperature,cloudiness, humidity,wind speed. The site provide the source data and Visualizations created as a part of analysis as well as explanations and descriptions of any trends and correlations witnessed.
